# Crafting

## Cách Dùng

```
/craft list              ← Xem tất cả công thức (hiển thị bạn có đủ nguyên liệu không)
/craft make <id>         ← Tổng hợp theo ID công thức
/craft materials         ← Xem kho nguyên liệu hiện tại
```

Cũng có thể craft trực tiếp trên **Dashboard web** (mục Chế Tạo).

### Targeted (chọn main stat cụ thể)

Thêm `_<stat>` vào cuối ID generic để craft với main stat cố định. Chi phí giống hệt generic cùng tier.

**8 stat có thể chọn:**

| Suffix | Main Stat |
|---|---|
| `_atk` | ★ ATK |
| `_crit` | ★ CRIT% |
| `_critDmg` | ★ CDMG% |
| `_eva` | ★ EVA% |
| `_pen` | ★ PEN% |
| `_resist` | ★ RES% |
| `_hp` | ★ HP |
| `_mpRegen` | ★ MP/TRN |

**Ví dụ:**

| ID | Item |
|---|---|
| `magic_c5_crit` | 🔵 Trang Sức [CRIT%] Magic Lv5 |
| `epic_c20_atk` | 🟠 Trang Sức [ATK] Epic Lv20 |
| `ancient_c40_pen` | 🔴 Trang Sức [PEN%] Ancient Lv40 |

> Sub stats vẫn random như bình thường — chỉ main stat được cố định.

## 🗡️ Tay Phụ (Offhand) — 3 loại: Khiên / Dao Phụ / Cầu Phép

Offhand có **3 loại** với main stat và sub stat pool khác nhau. Mỗi tier sinh ra 3 recipe tương ứng.

- 🛡️ **Khiên** (shield) — main stat DEF, yêu cầu STR
- 🗡️ **Dao Phụ** (dagger) — main stat DEX, yêu cầu DEX

> Offhand **không drop** — chỉ craft hoặc mua từ chợ người chơi.
