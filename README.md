# DautoRPG Wiki

Chào mừng đến với **DautoRPG** — game nhập vai tự động trên Discord!

## Tính Năng Chính

| Tính năng | Mô tả |
|---|---|
| ⚔️ **3 Class** | Chiến Binh, Thích Khách, Pháp Sư |
| 🗺️ **11 Vùng Explore** | Từ Rừng Xanh đến Thiên Đường (Lv 1–80) |
| 🏛️ **8 Tier Dungeon** | Boss, loot, nguyên liệu quý hiếm |
| 🎒 **5 Rarity** | Common → Magic → Rare → Epic → Ancient |
| 🔨 **Crafting** | Tổng hợp đồ từ nguyên liệu, rã/ghép nguyên liệu |
| ✨ **Phong Ấn** | Tăng chỉ số trang bị lên tới +5 |
| 🏪 **Chợ** | Mua bán đồ giữa người chơi |
| 👹 **Raid Boss** | Boss cộng đồng, phần thưởng theo đóng góp |
| 📋 **Nhiệm Vụ** | Quest hàng ngày reset lúc 00:00 UTC |

## Bắt Đầu Nhanh

```
/start <class> [tên]   ← Tạo nhân vật
/explore forest        ← Đi phiêu lưu đầu tiên
/status                ← Xem chỉ số nhân vật
/help                  ← Xem tất cả lệnh
```
