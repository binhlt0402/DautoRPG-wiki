# Khởi Động Nhanh

## 1. Tạo Nhân Vật

```
/start <class> [tên]
```

Chọn một trong 3 class và đặt tên nhân vật (tùy chọn). Mỗi tài khoản Discord chỉ có **1 nhân vật**.

→ Xem chi tiết từng class tại [Chọn Class](classes.md)

## 2. Đi Phiêu Lưu

```
/explore forest
```

Explore tự động chiến đấu, nhặt EXP, vàng và nguyên liệu. Mỗi lần explore tối đa **100 lượt** (6 giây/lượt).

## 3. Xem Chỉ Số

```
/status
```

Hiển thị HP, MP, ATK, DEF, EXP, vàng, trang bị đang mặc và Power Rating.

## 4. Mua Trang Bị Đầu Tiên

```
/shop list
/shop buy <đồ>
/equip <id>
```

Shop bán đồ ⚪ Common. Kiểm tra ID trong `/inventory list` rồi dùng `/equip` để trang bị.

## 5. Lên Dungeon

Khi đủ mạnh, vào Dungeon để nhặt đồ Rare/Epic và nguyên liệu quý:

```
/dungeon 1
```

## Vòng Lặp Cơ Bản

```
Explore → EXP/Vàng → Lên cấp → Mua/Craft đồ → Dungeon → Đồ tốt hơn → Explore vùng cao hơn
```

## Tips Cho Người Mới

* Dùng `/alloc auto` để tự động phân bổ điểm cấp vào main stat
* Dùng `/autoequip` để bot tự mặc đồ tốt nhất
* Cất đồ quý vào `/chest deposit` để tránh bán nhầm
* Xem nhiệm vụ hàng ngày bằng `/quest` để nhận thưởng thêm
